Scatterplot

### Prepare the data
x <- voteshare$group  # Voting methods
y <- voteshare$Trump  # Trump's votes

par(las = 1, mar = c(4, 4, 2, 4), cex = 0.8)
plot(
  1:length(y), y,
  type = "b",                
  pch = 16,                  
  col = "blue",              
  xaxt = "n",                
  xlab = "Voting Methods",   
  ylab = "Trump's Voteshare", 
  main = "Trump's Voteshare Across Voting Methods"
)

axis(1, at = 1:length(x), labels = x)

grid(nx = NA, ny = NULL, col = "gray", lty = "dotted")

text(
  1:length(y), y,
  labels = paste0(y, " "),
  pos = 4, cex = 0.8, col = "blue"
)

# Load ggplot2 library
library(ggplot2)

# Boxplot for Trump's vote share by voting method (group)
boxplot <- ggplot(voteshare, aes(x = group, y = Trump)) + 
  geom_boxplot(fill = "lightblue", color = "black", width = 0.5) +  # Classic boxplot style with whiskers and outliers
  labs(title = "Trump's Vote Share by Vote Type", x = "Vote Type", y = "Vote Share") +  # Labels
  theme_minimal()  # Simple, clean theme without being too fancy

ggsave("boxplot.jpg", plot = boxplot, width = 8, height = 6, dpi = 300)

The differences in these file formats are that, PDF files tend to look larger, more legible and professional. But PDFs are not image-type files by default. Pdfs can also be fillable, that is, they can operate as alsmost a word document with a more professional aesthetic and a format that doesn’t change.JPGs are image-type formats that are smaller in size than PDFs and have defined borders. They are often used for photos. SVGs are also image-type files, and their small file sizes make them useful for mobile viewing.
